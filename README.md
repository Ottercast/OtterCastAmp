# OtterCastAmp

OtterCastAmp is an open-source Sonos Amp replacement, based on a Sochip S3 SoC and pulseaudio sink/source.

##### Note: DO NOT ORDER/PRODUCE - it still got bugs, this note will disappear when fixed.
